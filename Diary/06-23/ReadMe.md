- 오늘 한 일 세줄 요약
```
    1. 1장 완성하기
    2. 3장 완성하기
    3. 마무리 짓기
```

- 이번주 작업 시간

```
    취업 관련 이슈로
    수업시간에 작업하였다
```



- - -

이번주 작업하고 다음주가 최종 발표이기 때문에  
급하더라도 마무리를 짓게 되었습니다.

먼저 1장 부분은 큰 내용 추가보다는 디테일 수정과 오타 수정에 노력하였고,  
3장 부분은 저번주에 목표로 한 부분 내용 추가와 급한 마무리를 짓는 작업을 하였습니다.
